## Gate State Machine

| Gate | Agents | Pause Status | Judge Action |
|---|---|---|---|
| Gate 1 — Intake | case-processing, complexity-routing | `awaiting_review_gate1` | Approve → Gate 2, or Re-run agent |
| Gate 2 — Dossier | evidence-analysis, fact-reconstruction, witness-analysis, legal-knowledge | `awaiting_review_gate2` | Approve → Gate 3, or Re-run agent |
| Gate 3 — Arguments | argument-construction, hearing-analysis | `awaiting_review_gate3` | Approve → Gate 4, or Re-run agent |
| Gate 4 — Verdict | hearing-governance | `awaiting_review_gate4` | Proceed to Decision (record `judicial_decision`) |
